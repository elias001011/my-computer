# Roadmap

## Phase 0 - Foundation

- Establish the repo, documentation, and product framing.
- Create a local server and web dashboard prototype.
- Store state locally in a JSON file.
- Make shell execution opt-in and off by default.
- Prepare the repo for future packaging, testing, and deployment.

## Phase 1 - MVP

- Persistent sessions.
- Chat flow with pluggable AI provider adapters.
- Terminal command execution with streaming output.
- Visible command history and event logs.
- Provider configuration for local or OpenAI-compatible endpoints.
- Environment variable management from the UI.
- Basic safety confirmations for sensitive actions.
- Install script and onboarding documentation.

## Phase 2 - Extensibility

- Skill manifests with permissions.
- Tool registration and validation.
- Multiple concurrent sessions.
- Scheduled tasks.
- Better configuration UX for local services and the agent.

## Phase 3 - Advanced Capabilities

- Secure remote access.
- Voice input and output.
- Multimodal video understanding.
- Desktop automation.
- Camera integrations and monitoring workflows.

## Phase 4 - Ecosystem

- Skill library or marketplace.
- Reusable setup profiles.
- Multi-machine support.
- Power-user workflows for builders and operators.
